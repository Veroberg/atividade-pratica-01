# Projeto Python - Análise de Dados

## Sobre o Projeto
Análise de datasets de e-commerce utilizando Python e Pandas para demonstrar habilidades em data analysis.

## Tecnologias Utilizadas
- Python
- Pandas
- Jupyter Notebook

## Estrutura de Dados
- `clientes.csv` - Cadastro de clientes
- `pedidos.csv` - Histórico de pedidos  
- `produtos.csv` - Catálogo de produtos

## Objetivos
- Análise exploratória de dados
- Identificação de padrões de compra
- Geração de insights estratégicos
